Pour `scores`, soit lire un seul champ `scoresJson` (string JSON.parse'd), soit collecter toutes les entrées dont la clé commence par `score__` et reconstruire `{ [criterionKey]: number }`. **Choisir l'approche `scoresJson`** (un input hidden côté form, plus simple à valider).

   i. **Check unique applicatif** (avant insert pour message lisible) : `select id from evaluations where submission_id=submissionId and evaluator_id=user.id`. Si existant → `{ ok:false, message:"Vous avez deja evalue cette soumission." }`.

      ```typescript
      .from("evaluations").insert({
      ```
