- `toCsv`: derive columns from `columns` arg or from keys of first row. Quote fields containing `,` `"` `\n` or `\r` ; double-quote escapes inner double-quotes (RFC 4180). null/undefined -> empty string. Numbers -> `String(n)`. Newline = `\r\n`.
- `csvResponse`: returns `new Response(body, { headers: { "Content-Type": "text/csv; charset=utf-8", "Content-Disposition": `attachment; filename="${filename}"`, "Cache-Control": "no-store" } })`.
- Add 4 inline assertion test cases at bottom under `if (require.main === module)` (per project convention since no test runner): empty rows -> empty string ; row with comma in value gets quoted ; row with quote `He said "hi"` gets escaped to `"He said ""hi"""` ; null becomes empty.

Create `app/admin/export/players.csv/route.ts`:

```typescript
import { NextResponse } from "next/server";
import { getCurrentRole } from "@/lib/auth";
import { getPlayersExportRows } from "@/lib/admin-export";
import { toCsv, csvResponse } from "@/lib/csv";

export const dynamic = "force-dynamic";

export async function GET() {
  const role = await getCurrentRole();
  if (role !== "game_master") {
    return new NextResponse("Forbidden", { status: 403 });
  }
  const rows = await getPlayersExportRows();
  const columns = [
    "team_slug", "team_name", "current_level", "status",
    "score_project", "score_engagement",
    "submissions_count", "validated_count",
    "leader_email", "member_emails",
  ];
  // Always include the header row even when rows is empty (demo / empty cohort).
  const headerOnly = rows.length === 0
    ? columns.join(",") + "\r\n"
    : toCsv(rows as unknown as Array<Record<string, string | number | null | undefined>>, columns);
  return csvResponse("players.csv", headerOnly);
}
```

Notes:
- Folder name with `.csv` is intentional — Next.js treats it as a literal segment.
- `dynamic = "force-dynamic"` because the route reads cookies (auth) and live DB.
- 403 (not redirect) is correct for an API/file download; non-GM users hitting this URL get a clear error rather than an HTML login page in their downloads folder.
