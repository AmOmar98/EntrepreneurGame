Creer `lib/results.ts` :

```typescript
export type RankingRow = {
  rank: number;
  player: Player;
  pitchAvg: number;       // moyenne arithmetique des PitchScore.total_score sur 100 ; 0 si aucun jure
  pitchJurorCount: number;
  scoreProject: number;   // copie de player.score_project
  combined: number;       // 0.5 * pitchAvg + 0.5 * scoreProject (par defaut)
};

export const DEFAULT_PITCH_WEIGHT = 0.5; // ponderation pitch ; project = 1 - pitch

export async function computeRanking(opts?: { pitchWeight?: number }): Promise<{
  eventId: string | null;
  publishedAt: string | null;
  rows: RankingRow[];
}>;

export async function isResultsPublished(): Promise<{ eventId: string | null; publishedAt: string | null }>;
```

Implementation `computeRanking` :
1. createClient(); null -> `{ eventId: null, publishedAt: null, rows: [] }`.
2. Resoudre event courant (id + results_published_at).
3. cohorts of event -> players (id, slug, name, idea, current_level, status, score_project, score_engagement, onboarded_at, cohort_id) ordre name asc.
4. Charger pitch_scores (player_id, total_score) WHERE event_id = eventId.
5. Aggregate cote serveur : pour chaque player_id -> moyenne(total_score) sur jurors qui ont score, count.
6. Calcul combined = pitchWeight * pitchAvg + (1-pitchWeight) * scoreProject. Default pitchWeight = 0.5.
7. Tri par `combined` desc, tie-break par `pitchAvg` desc puis `player.name` asc.
8. Assigner `rank` (1-based, dense ranking - egalite -> meme rang, prochain rang skip).

`isResultsPublished` : query events (id, results_published_at) du dernier event ; retourner `{ eventId, publishedAt }`.

Tous les console.error sur Supabase errors. Numeric(6,2) -> Number().
