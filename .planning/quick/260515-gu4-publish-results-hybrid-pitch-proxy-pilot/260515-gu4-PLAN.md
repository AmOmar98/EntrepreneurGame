### Étape 5 — Validation

Exécuter `SELECT * FROM lib/results.computeRanking()` équivalent SQL :

```sql
WITH event AS (SELECT id FROM events WHERE slug='agreentech-fes-meknes-mai-2026'),
pitch AS (
  SELECT ps.player_id,
    AVG(CASE WHEN ps.c5>0 THEN ps.total_score ELSE ps.total_score*1.25 END) AS pitch_avg
  FROM pitch_scores ps JOIN event e ON ps.event_id=e.id
  GROUP BY ps.player_id
)
SELECT p.slug, p.name,
  COALESCE(pi.pitch_avg, 0) AS pitch_avg,
  p.score_project,
  0.8*COALESCE(pi.pitch_avg,0) + 0.2*p.score_project AS combined
FROM players p
LEFT JOIN pitch ON pitch.player_id=p.id
ORDER BY combined DESC, pitch_avg DESC, p.name;
```
